# Setup

- Get an api key from [https://www.themoviedb.org/](https://www.themoviedb.org/)
- Create a .env file on the root directory with:

```jsx
REACT_APP_SERIE_API=(YOUR API KEY)
```

- Execute the following commands

- For npm
```jsx
npm install
npm start
```

- For yarn
```jsx
yarn
yarn start
```
